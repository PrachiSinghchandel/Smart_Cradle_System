# Smart_Cradle_System
An IoT-based baby monitoring system using a PIR sensor for motion detection. Detected movement is processed by the microcontroller and real-time alert notifications are sent to parents via the Blynk IoT platform over Wi-Fi. The system enables remote monitoring and improves infant safety through instant motion alerts.
